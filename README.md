# pinit
## an application for creating shortcut for apps and scripts ( only for linux )
![icon](images/pinit.svg)
## installation
- using pip :
run this two commands in terminal :
```bash
pip install pinit
pinit install
```

or run :
```bash
pip install pinit && pinit install
```
